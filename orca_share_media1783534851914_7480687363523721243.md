# 中阿桥 Zhong-A Bridge Landing Page

## 🚀 رفع الموقع على GitHub Pages (خطوات سهلة)

### الخطوة 1: أنشئ Repository جديد
1. ادخل على [github.com/new](https://github.com/new)
2. اسم الـ Repository: `zhong-a-bridge` (أو أي اسم تحبه)
3. اختر **Public**
4. اضغط **Create repository**

### الخطوة 2: رفع الملفات
```bash
# في جهازك، افتح Terminal/CMD واكتب:

git clone https://github.com/YOUR_USERNAME/zhong-a-bridge.git
cd zhong-a-bridge

# انسخ ملف index.html اللي عملته
# (حط الملف في الفولدر ده)

git add .
git commit -m "Initial landing page"
git push origin main
```

### أو الطريقة السهلة (بدون Terminal):
1. ادخل على Repository اللي عملته
2. اضغط **Add file** → **Upload files**
3. اسحب ملف `index.html` وحطه
4. اضغط **Commit changes**

### الخطوة 3: فعل GitHub Pages
1. في الـ Repository، اذهب لـ **Settings**
2. من الشمال، اضغط **Pages**
3. في **Source**، اختر **Deploy from a branch**
4. في **Branch**، اختر **main** و **/ (root)**
5. اضغط **Save**

### الخطوة 4: شوف موقعك! 🎉
- بعد 1-2 دقيقة، موقعك هيكون live على:
  `https://YOUR_USERNAME.github.io/zhong-a-bridge`

## 🎨 تخصيص الموقع

### تغيير الألوان
افتح `index.html` ودور على `:root` في الـ CSS:
```css
:root {
    --chinese-red: #E63946;   /* الأحمر الصيني */
    --arabic-blue: #1D3557;   /* الأزرق العربي */
    --gold: #F4A261;          /* الذهبي */
    --green: #2a9d8f;         /* الأخضر */
}
```

### تغيير الـ Waitlist Count
دور على هذا السطر في الـ JavaScript:
```javascript
let count=1247; // غير الرقم ده
```

### تغيير الـ Testimonials
دور على `testimonials-section` في الـ HTML وعدل الأسماء والتعليقات.

### إضافة Google Analytics
أضف هذا الكود قبل `</head>`:
```html
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```
